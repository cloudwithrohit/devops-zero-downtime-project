# Zero Downtime Deployment Project 🚀

## Project Overview
This project demonstrates a complete DevOps CI/CD pipeline using AWS, Jenkins, Docker, GitHub, and Nginx with Zero Downtime Deployment concepts.

---

## Tools & Technologies Used
- AWS EC2
- Jenkins
- Docker
- Docker Hub
- GitHub
- Nginx
- Linux

---

## Features
- CI/CD Pipeline Automation
- Docker Container Deployment
- Jenkins Integration
- GitHub Integration
- DockerHub Push Automation
- Nginx Reverse Proxy
- Blue-Green Deployment Concept
- Zero Downtime Deployment

---

## CI/CD Workflow
GitHub → Jenkins → Docker Build → DockerHub Push → Deploy Container → Nginx

---

## Steps Performed
1. Created AWS EC2 Instance
2. Installed Jenkins & Docker
3. Configured Jenkins Pipeline
4. Integrated GitHub Repository
5. Built Docker Images
6. Pushed Images to DockerHub
7. Deployed Containers using Docker
8. Configured Nginx Reverse Proxy
9. Implemented Zero Downtime Deployment Concept
